# Data_Comparative_Assessment_of_NL_Analysis_Software_for_-PBD_of_RC_Buildings_Part_1

Dataset Title: Comparative Assessment of Nonlinear Analysis Software for Performance-Based Design of Reinforced Concrete Buildings - Part 1: Component-Level Validation

Authors: Saman A. Abdullah; Jeff Dragovich; Kristijan Kolozvari; Andrés Lepage

Description

This dataset contains the input data and models used in the validations reported in the paper “Comparative Assessment of Nonlinear Analysis Software for Performance-Based Design of Reinforced Concrete Buildings - Part 1: Component Level Validation,” submitted to Earthquake Spectra (2026).
2D-dimensional nonlinear element models were developed in ETABS, PERFORM 3D and OpenSees to capture nonlinear behavior of concrete coupling beams and walls to validate nonlinear material and element models available in these software applications and to conduct systematic comparison between the three software. The dataset enables replication of the analyses results comparing the considered software.

Folder Structure
1.	Coupling Beams
a)	ETABS: Contains ETABS models of the various coupling beam types used in the study.
b)	OpenSees: Contains OpenSees input files of the various coupling beam types used in the study.
c)	PERFORM3D: Contains PERFORM3D models of the various coupling beam types used in the study.
2.	Walls
a)	ETABS: Contains ETABS models of the two wall specimens used in the study.
b)	OpenSees: Contains OpenSees input files of the two wall specimens used in the study.
c)	PERFORM3D: Contains PERFORM3D models of the two wall specimens used in the study.
